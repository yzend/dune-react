# formatLargeNumber

### 功能描述

将大数字格式化为更易读的形式，支持自定义单位和格式化配置。

### 参数说明

<ParamsTable
  params={[
    {
      name: "num",
      type: "number",
      required: true,
      description: "要格式化的数字",
    },
    {
      name: "format",
      type: "object",
      detailInfo: JSON.stringify({
        unit: 10000,
        unitCN: "万",
      }),
      required: false,
      description: "格式化配置对象",
    },
  ]}
/>

### 返回值

- **类型**: string
- **描述**: 格式化后的字符串

### 使用示例

```typescript
import { formatLargeNumber } from "./utils/utils-api";

// 基础用法
formatLargeNumber(15000);
// 结果: "1万+"

formatLargeNumber(5000);
// 结果: "10000+"

// 自定义配置
formatLargeNumber(1000000, { unit: 1000000, unitCN: "百万" });
// 结果: "1百万+"

formatLargeNumber(500000, { unit: 1000000, unitCN: "百万" });
// 结果: "1000000+"

// 处理边界情况
formatLargeNumber(0);
// 结果: "10000+"

formatLargeNumber(9999);
// 结果: "10000+"
```

### 特性

- ✅ 自动处理小于单位的数字
- ✅ 支持自定义单位和标识
- ✅ 向下取整处理
- ✅ 自动添加 "+" 后缀
